HTML Elements

Structural Elements

  • html - container element for entire document
  • head - for meta tags, title for tab, links to css, icons, javascript
  • basic structure - body contains everything use nav, sections & footer to group things instead of just divs give pages strucuture

Commonly Used Elements

  • divs - general containers
  • p - paragraphs or general text with default margins in place
  • a - anchor tags used to link to other files, do not use as a button
  • buttons - used for dynamic experiences with JavaScript or to submit forms
  • lists - ordered (numbered) lists & unordered lists

Semantic HTML5

nav and header used for navigation regions

main used for the area

footer is used for the bottom area
